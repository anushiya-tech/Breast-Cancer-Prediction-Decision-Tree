Breast Cancer Diagnosis Prediction
       This project uses machine learning in R to classify breast cancer tumors as Benign or Malignant. Two models—Decision Tree and Random Forest—are trained on a breast cancer dataset and evaluated using performance metrics such as accuracy, sensitivity, specificity, confusion matrix, and Kappa
       The models are compared, and the algorithm with the highest accuracy is automatically selected as the best model.
Technologies
   •R
   •caret
   •rpart
   •randomForest
Usage
Load the dataset in R, run the script, and the system will train both models, evaluate their performance, and display the best-performing model.
